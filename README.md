# Features
*    **Quran Text**

 *   **Audio Recitation**: Listen to the beautiful recitation of Quran verses 

  *  **Azkar Massa and Azkar Sabah**: Daily azkar to be recited in the morning and evening, *promoting a deeper connection with the Quran.*

 *   **Random Ayah**: Upon entering the app, users are greeted with a random Quran verse, fostering a sense of exploration and inspiration.

 *   **Tafseer Popup**: By tapping on any verse, users can access a popup displaying the corresponding tafseer (interpretation) to gain deeper insights into its meaning.
